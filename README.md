Building a Secure Corporate Network Infrastructure with Cisco Packet Tracer

I would like to share my next project where I applied my knowledge in the field of network engineering. The main goal of this project was to design a secure and efficient network topology that connects two different branches.

Key technical details of the project:

🔹 VLAN & Segmentation: I logically separated departments such as IT, HR, PERSON, SALES, and GUEST, and ensured communication between them (Inter-VLAN) using Router-on-a-Stick configuration.

🔹 Dynamic Routing (OSPF): I implemented the OSPF protocol to enable routers in both branches to automatically discover each other and select the most optimal path.

🔹 ACL (Access Control): To enhance network security, I applied Extended ACLs. Specifically, I restricted users to access only permitted resources and protected critical segments (IT block) from unauthorized access.

🔹 Secure Remote Management (SSH): I enabled encrypted SSH connections for managing network devices and limited access only to administrator networks (IT and SALES).

🔹 Automation: I simplified network management by configuring DHCP to automatically assign IP addresses to devices.

This project helped me gain valuable hands-on experience in packet flow, network design, and real-time troubleshooting.
